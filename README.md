# ChemReguCheck　(化学物質法規制チェッカー)

## 概要
これは、化学物質の物質名(日本語命名法、IUPAC命名法、SMILES記法)を入力することで、その物質が日本の法令で規制されているか判定することを目指したWebアプリケーションです。
この結果はあくまで簡易的な確認を行ったもので、"規制されているものを検出できない"あるいは"規制されていないものを検出する"場合があります。

## 主な機能
- **検索**:
  - 日本語名で入力すると、独自のロジックで英語に翻訳したのち、OPSINでSMILES(構造)に変換して検索します。
  - IUPAC名(英語)を入力すると、OPSINでSMILES(構造)に変換して検索します。
  - SMILES(化学構造を表す文字列)での直接検索も可能です。
- **法規制判定**: このプログラムにある法規制データベースと照合し、該当区分を表示します。
    - 法律名
    - 法律の該当する箇所
    - 該当する物質名
    - 該当した物質との一致レベル(完全一致、塩類など)
- **出力内容**:
    - 入力した物質名
    - 英語名 (入力が**英語**もしくは**SMILES**なら入力した物質名と同じもの、入力が**日本語**なら独自のロジックで英語に変換されたもの)
    - SMILES (入力が**SMILES**なら入力した物質名と同じもの、入力が**日本語**もしくは**英語**ならOPSINで変換されたもの)
    - 検出された法規制の一覧

## ディレクトリ構成
```
.
├── app.py                  # メインアプリケーション
├── ja_to_smiles.py         # 翻訳ロジック (日本語 -> 英語 -> SMILES)
├── checker.py              # 法規制判定ロジック
├── libs/
│   └── opsin-cli-2.8.0-jar-with-dependencies.jar # OPSIN (IUPAC変換ツール)
├── static/
│   ├── style.css           # デザイン
│   ├── script.js           # フロントエンド処理
│   ├── laws.txt            # 対応法令一覧テキスト
│   └── news.txt            # 更新情報テキスト
├── templates/
│   └── index.html          # 検索画面
├── dicts/              # 英語翻訳のための辞書データベース
│   ├──cores_dict.json       # 母核・元素など
│   ├──modifiers_dict.json  # 修飾語・官能基
│   ├──prefixes_dict.json   # 接頭辞 (モノ, ジ,など)
│   ├──suffixes_dict.json   # 接尾辞・塩 (塩化, 酸,など)
│   └──synonyms_dict.json   # 表記揺れ・慣用名変換
├── laws/               # 法規制データベース
│   ├── law1.json          # 法律A
│   └── law2.json          # 法律B
├── requirements.txt        # 依存ライブラリ一覧
├── THIRD_PARTY_LICENSES.txt # 第三者ライセンス表示
├── LICENSE                 # 本アプリのライセンス
└── README.md               # 説明書
```
